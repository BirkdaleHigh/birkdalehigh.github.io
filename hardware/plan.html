Hardware

Monday, Wednesday after school in U05 for years 9, 10, 11

The Plan

The hardware club is going to tie together everything you need to know to make a real, physical button pressed on a table change the display on a different computers screen.

Goals

Create some useful project for other clubs or areas of the school. Using the web, stand alone or hardware projects.

Skills

Through this club you will get to learn how to;

  • Create Servers
  • Create Clients
  • Create Artwork
  • Program Hardware
  • Handle User Interaction
